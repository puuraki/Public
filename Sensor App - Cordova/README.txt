Android, Cordova, JavaScript, Bootstrap
This sensor app was made using Cordova. The app uses bootstrap library and it contains four tabs: Home, Compass, GPS, Map.
Home tab shows device information and a short description of the app.
Compass tab shows information from the compass sensor (magnetic heading, true heading and accuracy).
GPS tab shows location information along with accuracy and speed.
Map tab contains a Google Map view of the current location.
